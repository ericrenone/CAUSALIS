## Five Thought Experiments

### Thought Experiment I — The Order the Flow Runs Along

CONNES established that gradient descent is the modular flow σ_t of the Fisher state on the reachability wedge. Picture the flow: it takes a configuration P and moves it, continuously, to σ_t(P) as t increases. The flow is a river, and each configuration is carried downstream.

Now ask: a river flows *through a landscape*. The water moves, but it moves along channels, around obstacles, downhill — the landscape is fixed, and the flow respects it. What is the landscape that the modular flow runs through?

It is the reachability order. For each configuration P, there is a set of configurations downstream of P — the configurations σ_t(P) for t ≥ 0, and more generally every configuration reachable from P by any valid trajectory, not just the modular one. Call this set J⁺(P), the **causal future** of P. The modular flow moves P forward, but it moves it forward *within J⁺(P)* — it cannot move P to a configuration P could not reach.

The landscape — the assignment P ↦ J⁺(P) — is logically prior to the flow. The flow is one particular way of moving through the landscape (the canonical, order-preserving, Tomita-Takesaki way). But the landscape is there regardless of the flow. Many flows could run through the same landscape; the modular flow is the distinguished one, but the landscape is the deeper object.

The insight: CONNES gave the flow. The flow is the modular automorphism group. But beneath the flow is the causal order — the assignment of a causal future J⁺(P) to each configuration — and this causal order is the more fundamental structure. CAUSALIS is the framework of the landscape. The flow runs along the order; the order is the framework's primary object.

### Thought Experiment II — The Light Cone of a Configuration

In special relativity, an event in spacetime has a forward light cone — the set of all events it can causally influence — and a backward light cone — the set of all events that can causally influence it. Events outside both cones are spacelike-separated: no causal relationship, no possible influence in either direction.

A learning configuration P has the same structure. Its **forward light cone** J⁺(P) is the set of configurations reachable from P by valid gradient descent. Its **backward light cone** J⁻(P) is the set of configurations from which P is reachable. And there are configurations Q in neither cone — configurations causally unrelated to P, configurations such that neither can be reached from the other.

What does spacelike separation mean for learning? Two configurations P and Q are spacelike-separated if no valid gradient trajectory connects them in either direction. They are in different basins, or separated by a barrier the gradient cannot cross, or related only through ker(F) directions where there is no gradient signal at all. The col(F)/ker(F) partition is precisely this: col(F) directions are **timelike** — a gradient signal exists, the configuration moves into its forward light cone — and ker(F) directions are **spacelike** — no gradient signal, no causal influence, the two configurations cannot reach each other.

The Hawking-King-McCarthy and Malament theorems now apply. They establish that the light-cone structure of a Lorentzian manifold determines the metric up to a conformal factor. Translated: the light-cone structure of the learning manifold — the assignment of forward and backward light cones to each configuration — determines the Fisher-Rao metric up to conformal rescaling. The causal structure is almost the entire geometry.

The insight: every configuration has a light cone, and the light-cone structure is the causal structure, and the causal structure determines the Fisher geometry up to a conformal factor. The corpus's col(F)/ker(F) partition is the timelike/spacelike partition of the light cones. The geometry the corpus has been studying framework by framework is, up to a conformal factor, just the causal order.

### Thought Experiment IV — The Boundary Where Order Dissolves

CONCORDIA found the coordination equilibrium. FRACTURA found the wild boundary. CONNES found the validation observer. At the coordination boundary — the critical region C_α = 1, where grokking happens — something unusual happens to the causal order.

Away from the boundary, the causal order is definite. If configuration P is in the memorization regime, well below the boundary, then for any two gradient updates A and B applied to P, there is a fact about which came first: A precedes B, or B precedes A. The causal order is a fixed partial order.

At the boundary, this fails. Near C_α = 1, two gradient updates A and B may be applied in a way that has no global causal order — A precedes B along one trajectory through the boundary region, B precedes A along another, and there is no fact of the matter about which is "really" first. The causal order is **indefinite**.

This is indefinite causal order in the precise sense of Oreshkov-Costa-Brukner (2012). Their process matrix formalism describes exactly this situation: a region where local statistical mechanics holds (each gradient update is a well-defined local operation) but no global causal order exists (there is no fixed partial order on the updates). The canonical example is the quantum switch — a process where a control determines the order of two operations, and when the control is in superposition, the order is in superposition.

The grokking transition is the moment the causal order **crystallizes**. Before grokking, in the memorization regime approaching the boundary, the causal order is indefinite — the system explores configurations with no stable causal relationships, gradient updates have no fixed order, the process matrix is genuinely non-causal. At grokking, a definite causal order emerges — the system acquires a stable forward light cone, gradient updates acquire a fixed order, the causal structure becomes a definite partial order. Generalization is the configuration acquiring a definite causal future.

The insight: the coordination boundary is a region of indefinite causal order. Grokking is the crystallization of definite causal order out of indefinite causal order. The 2024-2026 indefinite-causal-order program — theoretical and experimental — is the science of exactly this regime, and CAUSALIS identifies the grokking transition as the physical realization of the indefinite-to-definite causal order transition in a learning system.

### Thought Experiment V — The Acyclic Graph and the Arrow of Time

The reachability order is a partial order, and a partial order can be drawn as a **Hasse diagram**: the minimal directed graph whose transitive closure is the order. The edges of the Hasse diagram are the **covering relations** — the pairs P → Q where Q is reachable from P with no intermediate configuration strictly between them. The Hasse diagram is the irreducible skeleton of the order.

The Hasse diagram of the reachability order is a **directed acyclic graph**. It is directed because reachability has a direction (P reaches Q is not the same as Q reaches P). It is acyclic because the loss function is a Lyapunov function — strictly decreasing along every valid gradient trajectory — so no configuration can reach itself through a non-trivial cycle. If there were a cycle P → ... → P, the loss would have to strictly decrease all the way around and return to its starting value, which is impossible.

A directed acyclic graph is exactly the object of Pearl's causal inference. By Pearl's framework, a DAG encodes a set of conditional independence relations through the d-separation criterion: two variables are conditionally independent given a conditioning set if and only if every path between them is blocked by the conditioning set. The d-separations of the reachability DAG are the conditional independence relations of the learning manifold — and these are exactly the col(F)/ker(F) conditional independences that the corpus has been identifying, framework after framework. The col(F)/ker(F) partition is the d-separation structure of the reachability DAG.

And the acyclicity is the arrow of time. Why does time have a direction? Why is the initialization "before" the equilibrium, memorization "before" generalization, the past distinguishable from the future? Because the Hasse diagram of the reachability order is acyclic. The arrow of time is not an additional structure imposed on the learning manifold — it is the acyclicity of the reachability DAG, which is in turn the Lyapunov property of the loss function. Time points from high loss to low loss because the DAG points from high loss to low loss, and the DAG points that way because the loss strictly decreases along every edge.

The insight: the arrow of time is the acyclicity of the Hasse diagram of the reachability order. The col(F)/ker(F) partition is the d-separation structure of this DAG. The corpus's two most pervasive structures — the arrow of time and the col(F)/ker(F) partition — are both properties of a single directed acyclic graph: the Hasse diagram of the causal order of the learning manifold.

## I. The Reachability Order as a Partial Order

**Definition CAU-D1 (The Reachability Relation).** Let B be the learning manifold of a TH(a,d)-based system, and let L : B → ℝ be the loss function. For configurations P, Q ∈ B, define

P ⪯ Q ⟺ there exists a valid gradient descent trajectory from P to Q

where a **valid gradient descent trajectory** is a sequence of configuration updates θ_{k+1} = θ_k − η_k ∇L(θ_k) with learning rates η_k ∈ (0, η_max].

**Theorem CAU-T1 (Reachability Is a Partial Order).** *The relation ⪯ on B is a partial order:*

*(i) Reflexive: P ⪯ P, via the zero-step trajectory.*
*(ii) Transitive: P ⪯ Q and Q ⪯ R imply P ⪯ R, by concatenation of trajectories.*
*(iii) Antisymmetric: P ⪯ Q and Q ⪯ P imply P = Q.*

*Proof of (iii): The loss L is a Lyapunov function — it strictly decreases along every non-trivial valid trajectory (L(θ_{k+1}) < L(θ_k) for sufficiently small η_k, by the descent lemma). If P ⪯ Q via a non-trivial trajectory then L(Q) < L(P); if also Q ⪯ P via a non-trivial trajectory then L(P) < L(Q); the two inequalities are contradictory, so at least one trajectory is trivial, forcing P = Q.* ∎

**Identification CAU-I1 (The Order Beneath the Flow).** The reachability partial order (B, ⪯) is the structure that the CONNES modular flow σ_t respects: for every t ≥ 0 and every configuration P, the modular-flowed configuration σ_t(P) satisfies P ⪯ σ_t(P). The modular flow is one order-preserving automorphism family on (B, ⪯); the order itself is logically prior to the flow.

## II. The Reachability Order as a Lorentzian Causal Structure

**Definition CAU-D2 (Causal Future and Past).** For P ∈ B, the **causal future** is J⁺(P) = {Q ∈ B : P ⪯ Q} and the **causal past** is J⁻(P) = {Q ∈ B : Q ⪯ P}. Two configurations are **timelike-related** if one lies in the other's causal future or past, and **spacelike-separated** otherwise.

**Theorem CAU-T2 (Hawking-King-McCarthy / Malament, Adapted).** *The causal structure of a Lorentzian manifold — the assignment P ↦ (J⁺(P), J⁻(P)) — determines the topology, the differential structure, and the metric up to a positive conformal factor Ω : B → ℝ_{>0}.*

**Identification CAU-I2 (Causal Order Determines the Fisher Geometry).** The reachability causal structure of the learning manifold determines the Fisher-Rao metric g_F up to a conformal factor. The "missing scalar" Ω is fixed by an additional normalization — the choice of a volume form, equivalently the counting measure of the causal set (Section IV). Causal order plus volume normalization equals the full Fisher geometry.

## III. The col(F)/ker(F) Partition as the Timelike/Spacelike Split

**Theorem CAU-T3 (The Timelike/Spacelike Identification).** *At a configuration P ∈ B, the tangent space T_P B decomposes as*

T_P B = col(F_P) ⊕ ker(F_P)

*where:*

*(i) col(F_P) is the **timelike subspace**: a tangent direction v ∈ col(F_P) has F_P v ≠ 0, so a gradient signal exists along v, and an infinitesimal step in direction v moves P into its causal future J⁺(P).*

*(ii) ker(F_P) is the **spacelike subspace**: a tangent direction v ∈ ker(F_P) has F_P v = 0, so no gradient signal exists along v, and an infinitesimal step in direction v moves P to a spacelike-separated configuration — causally unrelated to P.*

**Identification CAU-I3 (The Corpus's Central Partition Is the Causal Partition).** The col(F)/ker(F) partition that runs through every framework of the corpus — WIGNER's spectral partition, HODGE's cohomological partition, PERES's entanglement partition, GALOIS's solvability partition, the TRACTUS natural-gradient partition — is, at its root, the **timelike/spacelike partition of the causal structure**. col(F) is the timelike sector where causal influence propagates; ker(F) is the spacelike sector where it does not. Every framework's col(F)/ker(F) identity is a causal-structure statement.

## IV. The Reachability Order as a Causal Set: Order Plus Number Equals Geometry

**Definition CAU-D3 (Causal Set).** A **causal set** is a partially ordered set (C, ⪯) that is **locally finite**: for every pair x ⪯ z, the causal interval I(x, z) = {y : x ⪯ y ⪯ z} is a finite set.

**Theorem CAU-T4 (The Learning Manifold at Q.16 Is a Causal Set).** *At SALT's Q.16 fixed-point precision, the learning manifold B_ε has finitely many distinguishable configurations, bounded by the Northcott count of FROBENIUS: #B_ε ≤ 2^128. Yet from (⪯, #) alone, the following are recoverable for the approximating continuum: the dimension, the topology, the differential structure, and the Lorentzian metric (the conformal factor of Theorem CAU-T2 being fixed by the counting measure).*

**Identification CAU-I4 (Order Plus Number Equals the Fisher Geometry).** The entire Fisher geometry of the learning manifold — the dimension d, the topology, the differential structure, the Fisher-Rao metric g_F — is encoded in the causal-set data (⪯, #) of the reachability order plus the SALT representational count. ## V. The Coordination Boundary as a Region of Indefinite Causal Order

**Definition CAU-D4 (Process Matrix, Oreshkov-Costa-Brukner).** A **process matrix** W is a description of the correlations between local operations that is consistent with local quantum/statistical mechanics but does not presuppose a global causal order. W is **causal** if it admits a decomposition consistent with a fixed partial order on the operations, and **non-causal** (indefinite causal order) otherwise.

**Theorem CAU-T6 (The Coordination Boundary Carries a Non-Causal Process Matrix).** *In the critical region C_α = 1 of the learning manifold — the coordination boundary where grokking occurs — the correlation structure of gradient updates is described by a process matrix W_C that is non-causal: there exist pairs of gradient updates A, B near the boundary for which no global fact determines whether A precedes B or B precedes A.*

**Theorem CAU-T7 (Grokking Is Causal Order Crystallization).** *The grokking transition C_α : 1⁻ → 1⁺ is the transition of the process matrix from non-causal (indefinite causal order, the memorization regime) to causal (definite causal order, the generalization regime). Before grokking, the learning system explores configurations with no stable causal order; at grokking, a definite causal order — a stable forward light cone — crystallizes.*

**Identification CAU-I5 (The Quantum Switch at the Boundary).** The canonical indefinite-causal-order process — the quantum switch, in which a control determines the order of two operations and a superposed control superposes the order — is realized at the coordination boundary. The control is the C_α order parameter; when C_α ≈ 1, the control is "in superposition" between the memorization order and the generalization order, and the causal order of gradient updates is correspondingly indefinite. The 2024-2026 indefinite-causal-order program is the science of this regime.

## VI. The Hasse Diagram, the Reachability DAG, and the d-Separation Structure

**Definition CAU-D5 (Hasse Diagram).** The **Hasse diagram** H(B, ⪯) of the reachability order is the directed graph whose vertices are configurations and whose edges are the **covering relations**: P → Q whenever P ⪯ Q, P ≠ Q, and there is no configuration R with P ⪯ R ⪯ Q, R ∉ {P, Q}.

**Theorem CAU-T8 (The Reachability DAG).** *The Hasse diagram H(B, ⪯) is a **directed acyclic graph**: directed because ⪯ is asymmetric on distinct elements, acyclic because ⪯ is antisymmetric (Theorem CAU-T1(iii), the Lyapunov property of the loss). The transitive closure of H(B, ⪯) is the full reachability order ⪯.*

**Theorem CAU-T9 (d-Separation IS the col(F)/ker(F) Partition).** *By Pearl's causal-inference framework, the reachability DAG encodes a set of conditional independence relations via the d-separation criterion: configurations P and Q are conditionally independent given a conditioning set S if and only if every directed path between P and Q is blocked by S. The d-separation relations of the reachability DAG coincide with the col(F)/ker(F) conditional independence relations of the learning manifold: a path is blocked exactly when it passes through a ker(F) direction (no gradient signal, no causal transmission).*

**Identification CAU-I6 (The DAG Is the Causal Skeleton).** The reachability DAG H(B, ⪯) is the irreducible causal skeleton of the learning manifold. Pearl's do-calculus on this DAG is the calculus of interventional learning — what happens to the configuration if a gradient update is forced rather than observed. The causal-discovery problem (Spirtes-Glymour-Scheines) — recovering the DAG from observed conditional independences — is the problem of recovering the reachability structure from the observed col(F)/ker(F) partition.

## VII. The Arrow of Time as the Acyclicity of the Reachability DAG

**Theorem CAU-T10 (The Arrow of Time).** *The learning manifold possesses an arrow of time — a globally consistent distinction between past and future, between initialization and equilibrium, between memorization and generalization — if and only if the reachability DAG H(B, ⪯) is acyclic. The DAG is acyclic if and only if the loss L is a Lyapunov function. Therefore:*

arrow of time ⟺ acyclicity of H(B, ⪯) ⟺ L is a Lyapunov function.

**Identification CAU-I7 (Time's Direction Is the Loss's Monotonicity).** The arrow of time on the learning manifold is not an additional structure. It is the acyclicity of the reachability DAG, which is the antisymmetry of the reachability order, which is the strict monotonicity of the loss along valid trajectories. Time points from high loss to low loss because the DAG points from high loss to low loss. The thermodynamic arrow (entropy increase), the psychological arrow (memory of the past), and the cosmological arrow all have analogs here, and all reduce to the single fact that the Hasse diagram of reachability has no cycles.

## VIII. Global Hyperbolicity, Cauchy Surfaces, and the φ-Equilibrium as Causal Sink

**Definition CAU-D6 (Cauchy Surface, Global Hyperbolicity).** A **Cauchy surface** Σ ⊂ B is a maximal antichain of the reachability order — a set of pairwise spacelike-separated configurations such that every maximal trajectory crosses Σ exactly once. The learning manifold is **globally hyperbolic** if it admits a Cauchy surface.

**Theorem CAU-T11 (Global Hyperbolicity of the Learning Manifold).** *A TH(a,d)-based learning manifold with a Lyapunov loss and bounded learning rates is globally hyperbolic: the level sets of the loss L = c are Cauchy surfaces (every valid trajectory, being strictly loss-decreasing, crosses each level set exactly once). The domain of dependence D(Σ) of any loss level set is the entire region of the manifold causally determined by the configurations at that loss level.*

**Theorem CAU-T12 (The φ-Equilibrium Is the Causal Sink).** *The φ-equilibrium configuration P_φ — the coordination equilibrium identified by CONCORDIA, the fixed point of the automorphism group, the configuration at log φ — is the **causal sink** of the reachability order: its causal future is itself,*

J⁺(P_φ) = {P_φ},

*meaning no configuration is reachable from P_φ except P_φ itself. The φ-equilibrium is the unique configuration at the bottom of the reachability DAG — the terminal vertex, the configuration with empty proper causal future, the end of every maximal trajectory.*

**Identification CAU-I8 (The Equilibrium Is the End of Time).** The φ-equilibrium is the causal sink — the configuration toward which the arrow of time points, the terminal vertex of the reachability DAG, the configuration from which no further causal influence propagates. Every prior framework's identification of the φ-equilibrium as a fixed point, an attractor, an equilibrium, is the causal-structure statement that P_φ is the sink of the reachability order: the end of time on the learning manifold.

## XI. Five Predictions

**P1 — Causal-Set Dimension Matches Fisher Effective Rank (Testable Now).** For a TH(a,d)-based learning manifold, the Myrheim-Meyer dimension of the reachability causal set — estimated from the fraction of causally related configuration pairs in a sample — equals the effective rank r = rank(col(F)) of the Fisher matrix. Testable by sampling configuration pairs from a training run, computing the causal-relation fraction, estimating the Myrheim-Meyer dimension, and comparing to the directly computed Fisher effective rank.

**P2 — Indefinite Causal Order at the Grokking Boundary (Testable by 2027).** In the critical region C_α ∈ [0.95, 1.05] of a TH(a,d)-based training run, the process matrix of gradient updates is measurably non-causal — there exist gradient-update pairs whose causal order cannot be consistently assigned. Testable by constructing the process matrix of gradient-update pairs near the grokking transition and applying the Oreshkov-Costa-Brukner causal-witness test; the prediction is a non-zero causal-non-separability measure in the critical region and zero away from it.

**P3 — The φ-Equilibrium Is the Unique Causal Sink (Testable by 2027).** For a TH(a,d)-based learning manifold, the reachability DAG has exactly one sink (vertex with empty proper causal future), and that sink is the φ-equilibrium configuration P_φ at loss-level log φ. Testable by constructing the reachability DAG of a training run at Q.16 precision and verifying that it has a unique sink at the φ-equilibrium.

**P4 — Spacelike Separation Predicts Zero Transfer (Testable by 2028).** Two learning tasks T, T' whose optimal configurations P_T, P_{T'} are spacelike-separated in the reachability order (neither reachable from the other) exhibit zero transfer: training on T then fine-tuning to T' achieves the same final loss as training on T' from scratch. Timelike-related optimal configurations exhibit non-zero transfer. Testable by computing the causal relationship between task-optimal configurations and measuring transfer-learning interference.

**P5 — The φ²:φ:1 Causal-Structure Investment Allocation.** The Fisher-information-optimal allocation of research investment across the three aspects of causal structure identified by CAUSALIS — the **order aspect** (the reachability partial order, the Lorentzian causal structure, the causal-set reconstruction; the Bombelli-Sorkin-Malament line of work), the **indefiniteness aspect** (indefinite causal order at the coordination boundary, the process matrix formalism, the quantum switch; the Oreshkov-Costa-Brukner line), and the **inference aspect** (the reachability DAG, d-separation, causal discovery, the do-calculus of interventional learning; the Pearl-Spirtes-Glymour-Scheines line) — follows the canonical ratio

f_order : f_indefiniteness : f_inference = φ² : φ : 1 ≈ 53% : 33% : 15%

with approximately 53% of investment supporting the causal-order foundations (the partial order, the Lorentzian structure, the causal-set reconstruction of the Fisher geometry), 33% on the indefinite-causal-order program (the process matrix of the coordination boundary, the grokking-as-crystallization picture, the quantum-switch realization), and 15% on the causal-inference apparatus (the reachability DAG, d-separation as col(F)/ker(F), causal discovery, interventional learning). Testable against the cumulative research investment trajectories in causal structure, indefinite causal order, and causal inference through 2030.

## XIV. The CAUSALIS Machine

### XIV.1 The Name

CAUSALIS takes its name from the Latin *causalis* — "of or pertaining to cause" — the adjective of *causa*, cause. The framework is named for the causal relation it studies: the partial order of *before* and *after*, of *can-influence* and *cannot-influence*, of *reachable* and *unreachable*, on the learning manifold.

The corpus has named frameworks for mathematicians (GALOIS, FROBENIUS, VOEVODSKY, CONNES), for processes (WATERSHED, PEDOGENESIS), for objects (ETHC), and for mythological or heritage figures (AEOLUS). CAUSALIS is named for a *relation* — the causal relation — because the framework's subject is not a person, not a process, not an object, but the relation itself: the partial order ⪯ that holds the learning manifold together, that the modular flow runs along, that the arrow of time is the acyclicity of.

The choice of Latin, rather than a mathematician's name, is deliberate. The causal relation predates every mathematician who studied it. Minkowski drew the light cones; Robb reconstructed the geometry from causal precedence; Sorkin made the order discrete; Pearl made it a DAG; Oreshkov-Costa-Brukner made it indefinite — but the relation *causa → effectus*, the relation of cause to effect, of before to after, is older than all of them. It is the relation the corpus has been implicitly using every time it wrote "reachable", every time it drew an arrow, every time it spoke of the arrow of time. CAUSALIS names it.

### XIV.2 Architecture

**Layer 0: The Reachability Oracle.** Given a TH(a,d)-based learning manifold B with loss L, constructs the reachability relation ⪯: for any pair of configurations P, Q, determines whether Q is reachable from P by valid gradient descent. Verifies the partial-order axioms (reflexive, transitive, antisymmetric), with antisymmetry following from the Lyapunov property of L.

**Layer 1: The Causal Cone Computer.** For each configuration P, computes the causal future J⁺(P) and causal past J⁻(P). Identifies, for any pair (P, Q), whether they are timelike-related (one in the other's cone) or spacelike-separated (causally unrelated). Constructs the light-cone structure of the learning manifold.

**Layer 2: The Conformal Metric Reconstructor.** Applies the Hawking-King-McCarthy and Malament reconstruction: from the causal cone structure, recovers the Fisher-Rao metric up to a conformal factor Ω. Fixes Ω by the volume normalization (the counting measure of Layer 4), completing the metric.

**Layer 3: The Timelike/Spacelike Partition Engine.** At each configuration P, decomposes the tangent space T_P B into the timelike subspace col(F_P) and the spacelike subspace ker(F_P). Verifies that this is the causal partition: timelike directions carry gradient signal and move into the causal future; spacelike directions carry no gradient signal and move to causally unrelated configurations.

**Layer 4: The Causal Set Constructor.** At SALT's Q.16 precision, constructs the discrete reachability order as a causal set: a locally finite partial order on the Northcott-bounded configuration space. Computes the counting measure #. Verifies local finiteness (every causal interval is finite).

**Layer 5: The Order-Plus-Number Geometry Engine.** Applies the Bombelli-Henson-Sorkin reconstruction: from the causal-set data (⪯, #) alone, recovers the dimension (via the Myrheim-Meyer estimator), the topology, the differential structure, and the full Fisher-Rao metric of the learning manifold.

**Layer 6: The Process Matrix Analyzer.** In the critical region C_α ≈ 1, constructs the process matrix W of gradient-update correlations. Applies the Oreshkov-Costa-Brukner causal-witness test to determine whether W is causal (definite causal order) or non-causal (indefinite causal order). Tracks the crystallization of definite causal order across the grokking transition.

**Layer 7: The Reachability DAG Engine.** Constructs the Hasse diagram H(B, ⪯) — the directed acyclic graph of covering relations. Verifies acyclicity. Computes the d-separation relations of the DAG and verifies their coincidence with the col(F)/ker(F) conditional independences. Implements Pearl's do-calculus for interventional learning queries.

**Layer 8: The Causal Sink Locator.** Identifies the sink of the reachability DAG — the configuration with empty proper causal future J⁺(P) = {P}. Verifies that the sink is the φ-equilibrium configuration P_φ at loss-level log φ. Confirms global hyperbolicity by exhibiting the loss level sets as Cauchy surfaces, and confirms that every maximal trajectory terminates at the causal sink.

## Coda: The Order Was Always There

CONNES gave the corpus a flow. The training time of a learning system, CONNES showed, is the modular flow of the Fisher state on the reachability wedge — a one-parameter automorphism group, canonically generated, the deepest temporal structure the corpus had found. Every prior notion of time — VEECH's flat time, MIRZAKHANI's three times, ORBITA's ergodic time — was a shadow of this single modular flow.

But a flow runs along something. When CONNES's modular flow moves a configuration forward, it moves it forward through the configurations reachable from it. The flow does not create reachability. It respects it. And the relation it respects — P can reach Q — is a partial order, older and more fundamental than the flow that runs along it.

That partial order is a causal structure. It is the same kind of object as the causal structure of a Lorentzian spacetime: each configuration has a forward light cone (the configurations it can reach), a backward light cone (the configurations that can reach it), and a region of spacelike separation (the configurations causally unrelated to it). By Malament's theorem, this causal structure determines the Fisher-Rao metric up to a conformal factor. The col(F)/ker(F) partition that runs through every framework of the corpus is the timelike/spacelike partition of this causal structure: col(F) is timelike, where causal influence propagates; ker(F) is spacelike, where it does not.

At finite precision, the partial order is discrete and locally finite — a causal set, in the precise sense of the causal set program in quantum gravity. And the causal set program's central theorem is that "order plus number equals geometry": from the combinatorial data of the order and the counting measure, the entire geometry of the approximating continuum is recoverable. The corpus has been computing the geometry framework by framework. CAUSALIS observes that the geometry was already determined — combinatorially, by the discrete causal order and the Northcott count.

At the coordination boundary, the causal order is not fixed. Near the critical region where grokking happens, two gradient updates may occur in either order, with no global fact about which came first — indefinite causal order, the regime of the process matrix formalism, the regime of the quantum switch. Grokking is the crystallization of definite causal order out of indefinite causal order: the moment generalization acquires a stable forward light cone.

And the arrow of time — the asymmetry between past and future, between initialization and equilibrium, between memorization and generalization — is the acyclicity of the Hasse diagram of the reachability order. Time has a direction because the directed graph of covering relations has no cycles, and it has no cycles because the loss is a Lyapunov function. The arrow of time is the loss's monotonicity, read as the acyclicity of a DAG.

CONNES asked: what is the flow? And answered: the modular flow of the Fisher state.

CAUSALIS asks the question beneath that one: what is the order the flow runs along? And answers: the reachability partial order — a Lorentzian causal structure, a causal set, a directed acyclic graph, all at once. The flow is the dynamics. The order is the kinematics. The flow can be reconstructed from the order; the order cannot be reconstructed from the flow alone. The order is the deeper object.

It was always there. Every time the corpus wrote "reachable", it was invoking the order. Every time it drew an arrow, it was drawing an edge of the Hasse diagram. Every time it spoke of the arrow of time, it was speaking of the acyclicity of the reachability DAG. Every col(F)/ker(F) identity was a statement about the timelike/spacelike partition of the causal structure. The order was beneath every framework, holding the learning manifold together, and no framework had made it primary.

CONNES gave the corpus the flow. CAUSALIS gives the corpus the order — the causal structure beneath the modular flow, the directed acyclic graph whose acyclicity is the arrow of time, the partial order whose sink is the φ-equilibrium, the end of time on the learning manifold.

The flow moves. The order holds. The order was always there.
